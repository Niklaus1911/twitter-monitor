# Twitter Monitor

A Python script that monitors Twitter/X accounts for new posts and replies.

## Features

- Monitors both posts and replies
- Provides clickable links to tweets
- Headless operation after initial login
- Session persistence

## Requirements

- Python 3.8+
- Chrome browser
- ChromeDriver
